# Cynthia | Cloud Engineer Portfolio

This is my personal cloud portfolio project built and deployed on AWS.

## 🚀 Project Overview
This project showcases a static portfolio website hosted using:
- Amazon S3 (static website hosting)
- Amazon CloudFront (CDN for global delivery and HTTPS)
- GitHub (version control)

## 🌐 Live Demo
[View Website](https://d1tpmg0rf53fgq.cloudfront.net)  

## 🛠️ Technologies Used
- HTML5
- CSS3
- AWS S3
- AWS CloudFront

## 📂 Project Structure
- `index.html` – Main webpage
- `style.css` – Styling for the website

## 📌 What I Learned
- Hosting static websites using S3
- Configuring bucket policies for public access
- Using CloudFront for secure content delivery (HTTPS)
- Connecting GitHub with deployment workflows
